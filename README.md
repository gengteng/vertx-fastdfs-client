The Vert.x FastDFS client provides an asynchronous API to interact with a FastDFS server.

Maven artifact:

```xml
<dependency>
    <groupId>org.gteng</groupId>
    <artifactId>vertx-fastdfs-client</artifactId>
    <version>4.2</version>
</dependency>
```
